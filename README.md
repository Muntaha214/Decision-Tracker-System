# DecisionOS

DecisionOS is an AI-powered productivity and decision support system that helps users determine what task they should work on next instead of acting as a simple to-do list.

## Planned Features

- Task Management
- Intelligent Decision Engine
- AI Recommendations
- Pomodoro Timer
- Delayed Task Detection
- Rewards & Streaks
- Analytics Dashboard
- Notifications
- Authentication
- Cloud Deployment

## Tech Stack

- Java
- Spring Boot
- HTML
- CSS
- JavaScript
- PostgreSQL
- REST APIs
- Gemini API
